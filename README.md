# 🐍 Python Mastery — Học Python Toàn Diện

Web app học Python với giao diện đẹp, đầy đủ nội dung từ cơ bản đến nâng cao.

## 📚 Nội dung

| Chương | Nội dung |
|--------|----------|
| 1 | Python Cơ Bản: Biến, Hàm, List, Dict, File |
| 2 | Hướng Đối Tượng (OOP) |
| 3 | Cấu Trúc Dữ Liệu & Giải Thuật |
| 4 | Tkinter GUI & SQLite Database |
| 5 | Matplotlib - Vẽ đồ thị |
| 6 | NumPy & Pandas - Data Science |

## 🚀 Chạy dự án

```bash
npm install
npm start
```

## 🌐 Deploy lên GitHub Pages

```bash
npm install --save-dev gh-pages

# Thêm vào package.json:
# "homepage": "https://USERNAME.github.io/python-mastery"
# "predeploy": "npm run build"
# "deploy": "gh-pages -d build"

npm run deploy
```
